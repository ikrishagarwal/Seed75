# Seed75
A completely hand made 75% mechanical keyboard from PCB to case, even the firmware.

![Seed75](/assets/keeb.png)

## Features
- 75% Layout
- Hot Swappable Sockets
- Sleep Design
- Customizable Push Knob
- USB C Port
- Detachable Angle Elevation
- Sandwich Mount
- Wrist Rest
